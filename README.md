# 🏋️ FitZone Gym – Fitness Website

A responsive and visually engaging single-page website for **FitZone Gym**. This project showcases a modern design with sections for gym services, contact info, and an interactive registration form.

## 🌟 Project Overview

FitZone Gym is designed to promote fitness services and allow users to register directly through the website. The landing page uses a full-screen background image, clean typography, and responsive layout to ensure compatibility across devices.

## 📁 Files Included

- `fitness.html` – Main HTML file containing the structure and style of the webpage.
- `fitnessimage.jpeg` – Background image used for visual appeal.
- *(Optional)* You can separate the CSS and JavaScript into individual files later for maintainability.

## 🧠 Features

- Smooth scrolling between sections
- Interactive registration form
- Background image integration
- Responsive footer with contact and hours
- Simple, intuitive navigation bar
